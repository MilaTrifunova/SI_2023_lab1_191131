4. $ mkdir SI_2023_lab1_191131 , со оваа команда е искреиран директориум(локален репозиториум).

$ cd SI_2023_lab1_191131 , со оваа команда ппристапуваме во креираниот локален репозиториум.

$ git init
Initialized empty Git repository in C:/Users/Mila/SI_2023_lab1_191131/.git/ , со оваа команда го иницијализираме Git во директориумот.

5. $ git remote add origin https://github.com/MilaTrifunova/SI_2023_lab1_191131.git , со оваа команда го поврзуваме локалниот репозиториум со оддалечениот репозиториум.

6. Се наоѓаме на master гранката и со оваа команда
$ nano README.md
се додава датотека и во неа запишуваме според барањето Mila Trifunova 191131.

$ git add README.md , со оваа команда се додадава датотетка, спремна за commit.

$ git commit -m "Initial commit" , со оваа команда се прави commit, на веќе подготвената датотека, со порака "Initial commit" .

7. $ nano SILab1.java , со оваа оманда се креира нов фајл во со име SILab1.java и во него ја прикачуваме ископираната содржина од линкот од документот за лабораториската.

$ git add SILab1.java , со оваа команда се додадава фајлот,  спремен за commit.

$ git commit -m "Add java class" , со оваа команда се прави commit, на веќе подготвениот фајл, со порака "Add java class" .

8. $ git push origin master , со оваа команда се сихронизираат сите гранки на локалниот со оддалечениот репозиториум.
